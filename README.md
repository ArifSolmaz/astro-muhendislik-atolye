## Yerel Çalıştırma

```bash
pip install numpy matplotlib scipy
python astro_muhendislik_atolye.py
```

Veya Jupyter ile:

```bash
pip install numpy matplotlib scipy jupyter
jupyter notebook astro_muhendislik_atolye.ipynb
```
